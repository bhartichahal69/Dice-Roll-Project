# 🎲 Dice Roll Project

## 📌 Description

This is a simple Python-based Dice Roll game.
The program simulates rolling a dice for players and displays the result. It is a beginner-friendly project to understand basic Python concepts like loops, functions, and random numbers.

---

## 🚀 Features

* Roll a dice for players
* Random number generation (1 to 6)
* Simple and interactive gameplay
* Beginner-friendly code

---

## 🛠️ Technologies Used

* Python

---

## ▶️ How to Run the Project

1. Install Python on your system
2. Download or clone this repository
3. Open the project folder in your code editor
4. Run the Python file:

```bash
python dice.py
```

## 🎮 How to Play

* Run the program
* Press the key to roll the dice
* The program will generate a random number between 1 and 6
* Players take turns

---

## 📸 Output Example

```
Player 1 rolled: 4
Player 2 rolled: 6
Player 2 Wins!
```
